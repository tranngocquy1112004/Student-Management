# Chat Services

This directory contains business logic services for the chat feature:

- `ChatService.js` - Main chat service for conversations and messages
- `PermissionService.js` - Permission validation for chat access

## Services Overview

### ChatService
- Conversation management (create, get, list)
- Message management (send, get, pagination)
- Unread count management

### PermissionService
- Permission matrix validation
- User list filtering by role
- Chat access control
