# Chat Components

This directory contains React components for the chat UI:

## Components

- `ChatLayout.js` - Main layout with sidebar and chat window
- `ConversationList.js` - List of conversations
- `ConversationItem.js` - Single conversation item
- `MessageList.js` - List of messages with infinite scroll
- `MessageItem.js` - Single message item
- `MessageInput.js` - Input for sending messages
- `TypingIndicator.js` - Shows when user is typing
- `OnlineStatus.js` - Shows online/offline status
- `UserSearch.js` - Search users to start conversation

## Styling

Each component has its own CSS file for styling.
